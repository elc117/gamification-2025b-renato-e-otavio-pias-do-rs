# Fact or Fake

## Classes Implementadas

Foram implementadas as seguintes classes para iniciar o projeto:

- **Usuario** — Representa um usuário do sistema com id, nome, email e nível.

- **Categoria** — Define categorias de notícias com nome, descrição, níveis totais, pontos para próximo nível e caminho da imagem completa.

- **Noticia** — Contém informações de uma notícia: título, conteúdo, se é verdadeira ou falsa, explicação, dificuldade e categoria associada.

- **Resposta** — Registra a resposta de um usuário para uma notícia, incluindo se está correta e pontos ganhos.

- **Conquista** — Define conquistas desbloqueáveis com nome, descrição, ícone, critério, tipo e valor requerido.

- **ConquistaUsuario** — Relaciona conquistas desbloqueadas por usuários com data de desbloqueio e status de visualização.

- **ProgressoCategoria** — Acompanha o progresso do usuário em cada categoria: nível atual, pontos de maestria e peças desbloqueadas.
